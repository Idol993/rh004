# 端云协同智能炒股分析系统 - Flutter 前端

## 📁 项目结构

```
smart_stock/dart/
├── lib/
│   ├── main.dart                    # 应用入口
│   ├── signal_models.dart           # 空值安全数据模型
│   ├── chart_config_generator.dart  # fl_chart 配置生成器
│   ├── stock_chart_page.dart        # 股票分析页面
│   ├── stock_data_service.dart      # 本地数据加载服务
│   └── mock_data.dart               # 备用内嵌模拟数据
├── assets/
│   └── demo_data/
│       ├── index.json               # 股票列表索引
│       ├── 000001.SZ.json           # 平安银行
│       ├── 600519.SH.json           # 贵州茅台
│       ├── 300750.SZ.json           # 宁德时代（无信号）
│       └── 601318.SH.json           # 中国平安（混合信号）
├── pubspec.yaml                     # 依赖配置
├── analysis_options.yaml            # 代码规范
├── .metadata                        # Flutter 工程元数据
└── SETUP_AND_RUN.md                 # 本文档
```

## 🚀 快速开始（3步运行）

### 1. 初始化 Flutter 工程

```bash
cd smart_stock/dart
flutter create .
```

### 2. 安装依赖

```bash
flutter pub get
```

### 3. 运行应用

```bash
# 连接模拟器或真机后
flutter run
```

## 📊 功能验证清单

运行应用后，你会看到完整的股票分析页面，包含以下联动功能：

### ✅ 顶部：股票切换下拉框
- 4 只股票可选：平安银行、贵州茅台、宁德时代、中国平安
- 切换后所有卡片同步刷新
- 加载中显示进度条，加载失败显示错误提示和重试按钮

### ✅ 预测卡片（第一部分）
- 趋势方向标签（看涨绿/看跌红/震荡橙）
- 预测收益率数值（带正负颜色）
- 趋势概率条形图（上涨/下跌/横盘）
- 置信区间三栏卡片（下限/上限/置信水平）

### ✅ 图表卡片（核心）
**5 种元素同时渲染：**

| 元素 | 样式 | 说明 |
|------|------|------|
| 历史收盘价 | 蓝色实线 | 80 个价格点 |
| 趋势预测 | 彩色虚线 | 接在 K 线后 10 个点，颜色随方向 |
| 买入信号 | 绿色上三角 ▲ | 标注在价格上方，大小随强度 |
| 卖出信号 | 红色下三角 ▼ | 标注在价格下方，大小随强度 |
| 注意力高亮 | 黄色垂直宽条 | 模型关注的连续时间段 |
| 置信区间 | 上下水平虚线 | 带"置信高/低"标签 |

### ✅ 信号卡片（第三部分）
- 买卖信号列表（图标+类型+位置+强度+价格）
- 宁德时代无信号时显示友好空状态（不崩溃）

### ✅ 注意力卡片（第四部分）
- 注意力热力图（蓝→橙渐变，左侧有灰色偏移填充对齐K线）
- 关键区间标签（黄色高亮胶囊）

## 🛡️ 空值安全验证场景

以下场景均已覆盖，页面不会崩溃：

| 场景 | 预期行为 |
|------|----------|
| `payload_json` 为空字符串 | 返回 `AnalysisPayload.empty()`，显示空数据 |
| `prediction` 字段缺失 | 所有预测字段使用默认值（概率=0，CI=[0,0]） |
| `signals: []` 空数组 | 信号卡片显示"当前无买卖信号" |
| `attention_weights: []` 空数组 | 注意力卡片显示"暂无注意力数据" |
| 某只股票 JSON 文件不存在 | 返回空对象，页面可操作，显示空状态 |
| JSON 格式非法 | `try-catch` 捕获，返回空对象 |

## 🔗 端到端联调流程

```
Python 后端
    ↓
generate_demo_data.py
    ↓ 生成
demo_data/*.json (5个文件)
    ↓ 自动同步到
dart/assets/demo_data/
    ↓ Flutter 运行时
rootBundle.loadString()  ← StockDataService
    ↓ 解析
AnalysisPayload.fromJson()  ← 空值安全
    ↓ 生成图表配置
ChartConfigGenerator.buildChartConfig()
    ↓ 渲染
LineChart() + 5种视觉元素
```
